# Student Result Analyzer
This is a user-friendly Python project that analyzes student results by
calculating total marks, average, and grade.

## Features
- Takes student names and marks as input
- Calculates total and average marks
- Assigns grades based on average
- Displays results in a clean format

- Grades system:
- average >= 90 then grade = 'A'
- average >= 75 then grade = 'B'
- average >= 60 then grade = 'C'
- Else grade = 'F'

## Technologies Used
- Python 3

## How to Run
1. Clone the repository
2. Open terminal in project folder
3. Run:
   python student_result_analyzer.py
